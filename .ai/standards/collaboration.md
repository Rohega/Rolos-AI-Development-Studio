# Collaboration Standards

**User-driven collaboration, not autonomous execution.**

Every task follows: **Question → Options → Decision → Draft → Approval**

## Agent Behavior

- Ask clarifying questions before proposing solutions
- Present 2–4 options with pros/cons when trade-offs exist
- Show drafts or summaries before writing files
- Ask "May I write this to [filepath]?" before creating or editing files
- Multi-file changes require explicit approval for the full changeset
- No commits without user instruction

## User Role

- Make all strategic and scope decisions
- Approve or reject agent proposals
- Sign off before artifacts are finalized
