# US-021: Picking list

**Spec:** [warehouse-mvp](../../specs/warehouse-mvp.md)  
**Priority:** P0  
**Dependencies:** US-020

---

## Goal

Generate a work list ordered by location so the picker walks through the warehouse efficiently when preparing an order.

## User

**Picker / Operator** (`operario`)

## Description

As an operator, I want to see a picking list with products, quantities, and locations ordered by aisle/rack/position, so I can prepare the outbound order by walking through the warehouse in a logical way.

## Acceptance criteria

- [ ] When picking starts, the system generates `picking_lines` assigning locations with available stock
- [ ] List ordered by `aisle`, `rack`, `position`
- [ ] Each line shows: product (SKU, name), location, quantity to pick, unit
- [ ] If a product needs several locations, it is split into multiple picking lines
- [ ] Shows progress: pending vs confirmed lines
- [ ] Product search in the view by SKU or name

## Business rules

1. MVP allocation strategy: **FIFO by location** (alphabetical order of location, consume the first one with stock).
2. Only locations from the order's warehouse.
3. Quantity to pick 竕､ available stock in the location (`on_hand - reserved` itself + this order's reservation).
4. No picking is generated for an order in `draft`.

## Edge cases

| Case | Expected behavior |
|------|-------------------------|
| Insufficient stock in a single location | Split across locations or mark the line as pending |
| Deactivated location with legacy stock | Exclude from new assignments |
| Regenerate picking after a stock change | Only if no line is confirmed; otherwise manual |
| Product in multiple locations in the same aisle | Order by position |

## Preliminary technical notes

- `PickingLine` model: `outbound_line_id`, `location_id`, `quantity_to_pick`, `quantity_picked`, `sequence`, `status`
- `Picking::GenerateLines.call(outbound_order)` service
- Greedy algorithm by ordered location
- Recommended ADR: allocation and reservation strategy
- Branch: `feature/warehouse-picking`
